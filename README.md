Carbon Emissions Awareness Project

Project Overview
This project was developed as part of a university initiative to create rapid awareness regarding climate change and individual carbon footprints. 

My Role: Tech Representative
As a volunteer tech representative, I spearheaded the technical side of this program to:
*   Develop a user-friendly platform to visualize carbon emission data.
*   Educate the community on the impact of rapid climate change.
*   Bridge the gap between technology and environmental sustainability.
